## 数据库环境搭建

## 启动MySQL服务

### Windows
1. 通过服务管理器启动：
   - 打开服务管理器（Win+R，输入services.msc）
   - 找到MySQL服务并点击"启动"
2. 命令行启动：
   ```
   net start mysql
   ```

### Linux
```
sudo systemctl start mysql
```

### macOS
```
brew services start mysql
```

## 导入SQL文件

### 前提条件
- 确保MySQL服务已启动
- 确保有SQL文件（.sql）
  - 文件所在目录：`nebula-framework/nebula-framework-core/src/main/resources/dump-nebula-202504032310.sql`

### 导入步骤
1. 打开命令提示符（Windows）或终端（Linux/macOS）
2. 进入SQL文件所在目录
3. 执行导入命令

### 通用方法（适用于所有平台）
```
mysql -u username -p database_name < dump-nebula-202504032310.sql
```

### Windows 
1. 使用DBeaver等工具导入 (**推荐**)
2. 使用MySQL Workbench图形界面导入
3. 在命令提示符中执行上述通用方法


### Linux/macOS
1. 使用DBeaver等工具导入 (**推荐**)
2. 使用MySQL Workbench图形界面导入
3. 在命令提示符中执行上述通用方法
